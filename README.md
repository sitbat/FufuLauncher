### 一个原神的第三方启动工具，支持游戏注入、自动签到和一些实用小功能。

![Image](https://youke2.picui.cn/s1/2025/12/16/6940312f3ce5d.png)

## 主要功能

1. **账号管理**：多账号快速切换，不用重复输入密码
2. **自动签到**：每天一键完成米游社签到
3. **游戏管理**：自动选择游戏路径，实时更新版本公告
4. **实用工具**：内置养成计算器、键盘连点器
5. **启动参数**：自定义分辨率、窗口模式等设置
6. **注入**：辅助体验更好的游戏内容

## 怎么用

1. 第一次打开会显示用户协议，同意后才能用，请仔细阅读它！
2. 去"设置"页面选一下游戏安装路径，最好同意自动识别的路径
3. 在"账号"页面登录米游社账号，回到主页就可以用签到了
4. 主界面点“点击启动游戏”就能启动游戏了

## 注意事项

1. 切换账号需要管理员权限
2. 游戏路径尽量别用管理员模式选，选完再管理员运行
3. 注入功能需要以管理员身份运行程序
4. 自定义背景支持图片和视频文件，动态视频背景不太稳定，图片正常

## 说明
- 请在使用项目前安装 **.NET 8.0** 或以上任意版本和 **Webview2** 运行时
- 本项目仅供学习交流使用，请支持官方正版游戏。
